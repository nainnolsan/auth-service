## Arquitectura

```
Frontend (React + Vite)
       │
       ▼
 API Gateway (GraphQL)
       │
       ▼
 Auth Service (REST)
       │
       ▼
  PostgreSQL
```

El **Frontend** solo habla con el **API Gateway** vía GraphQL. El API Gateway se encarga de comunicarse con los microservicios internos vía REST.

## Stack Tecnológico

| Capa         | Tecnología                          |
|--------------|-------------------------------------|
| Frontend     | React, TypeScript, Vite, Apollo     |
| API Gateway  | Node.js, Apollo Server, GraphQL     |
| Auth Service | Node.js, Express, TypeORM, JWT      |
| Base de Datos| PostgreSQL                          |
| Deploy       | Railway                             |
