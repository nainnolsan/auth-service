## POST /api/auth/login

Autentica a un usuario existente. Verifica las credenciales y devuelve un par de tokens JWT.

<Note>
  Por seguridad, cuando las credenciales son inválidas el mensaje de error **no especifica** si el email no existe o si la contraseña es incorrecta. Siempre responde con "Credenciales inválidas".
</Note>

## Request Body

<ParamField body="email" type="string" required>
  Email del usuario registrado.
</ParamField>

<ParamField body="password" type="string" required>
  Contraseña del usuario.
</ParamField>

## Responses

<ResponseField name="success" type="boolean">
  Indica si la operación fue exitosa.
</ResponseField>

<ResponseField name="message" type="string">
  Mensaje descriptivo del resultado.
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="data">
    <ResponseField name="accessToken" type="string">
      JWT de corta duración (15 min) para autenticar requests.
    </ResponseField>

    <ResponseField name="refreshToken" type="string">
      JWT de larga duración (7 días) para renovar el access token.
    </ResponseField>

    <ResponseField name="user" type="object">
      <Expandable title="user">
        <ResponseField name="id" type="string">
          UUID único del usuario.
        </ResponseField>
        <ResponseField name="name" type="string">
          Nombre del usuario.
        </ResponseField>
        <ResponseField name="email" type="string">
          Email del usuario.
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

## Ejemplos

<CodeGroup>

```bash cURL
curl -X POST https://tu-auth-service.railway.app/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "email": "juan@example.com",
    "password": "mipassword123"
  }'
```

```js JavaScript (fetch)
const response = await fetch('/api/auth/login', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    email: 'juan@example.com',
    password: 'mipassword123'
  })
});

const data = await response.json();
// Guardar tokens en localStorage o memory
```

```graphql GraphQL (via API Gateway)
mutation Login {
  login(email: "juan@example.com", password: "mipassword123") {
    accessToken
    refreshToken
    user {
      id
      name
      email
    }
  }
}
```

</CodeGroup>
